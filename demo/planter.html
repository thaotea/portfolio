Planter is a mobile application that makes taking care of your plants easier by helping to identify your plant and send reminders for care!

learn more










Planter is an application inspired by my sister who always forgot to water her plants and when she finally remembered, it was too late to save them. Planter helps the user figure out what type of plant it is as well as a short description for the plant's preferences and characteristics. The application has a simple interface that makes adding plants, setting the watering and fertilizing reminders, and checking them off daily quick and easy.


NAVIGATION

I started off with a plan for Planter's navigation to highlight and plan the application's core functionalities. On the homepage, I planned to include the list of the user's plants, a button for adding plants, and a button for settings. Each plant has a picture, name, checkboxes to check off watering and fertilizing if necessary, and an edit button. When a user wants to add a plant, the user could either identify their plant using the camera or look up the plant in the library. After a brief description of the plant, the user will input the schedule for the plant's watering and fertilizing. In the settings, the user can turn on and off push notifications and learn more about Planter.
