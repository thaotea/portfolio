PROJECT

Moneybook

MY ROLE

UI Designer
Front-end Developer

SUMMARY

I designed and developed a desktop application that simplifies the process of keeping track of your daily expenses.

DESKTOP DEMO

OVERVIEW

Moneybook makes keeping track of everyday expenses simple and easy. The application allows the user to keep track of their spending in individual categories so they can analyze where they're spending their money. The application features Google Map Charts so the user can physically see and budget their future finacial decisions. The user can adjust the date to see daily, weekly, and monthly spending throughout the year.

ENDING NOTES

Through this personal project, I gained hands-on experience working with an Google API for the first time. I was able to mainly strengthen my JavaScript skills to get the necessary features such as the calendar and the chart input. I wanted to imitate the experience of filling out an excel sheet to keep track of finances and make it more interactive.
